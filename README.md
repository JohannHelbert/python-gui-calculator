# 🧮 Python GUI Rechner

Ein einfacher grafischer Taschenrechner mit Python und Tkinter, der Grundrechenarten unterstützt – plus erweiterte Funktionen wie:

- Potenz (xʸ)
- Wurzel (√x)
- Vorzeichenwechsel (±)
- GUI mit modernem Design

## 🛠️ Voraussetzungen

- Python 3.x
- Tkinter (in Python enthalten)

## 🚀 Starten

```bash
python calculator.py
```
